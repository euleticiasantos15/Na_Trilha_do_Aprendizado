Tag iframe

Essa Tag é usada para incorporar outro documento ao documento HTML, esse outro documento pode ser: texto, áudio, imagens, animações ou vídeo em uma única apresentação.

Incorporar vídeos ao site

Tag vídeo

Essa tag é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo. A tag vídeo contém uma ou mais source tags com diferentes fontes de vídeo. O navegador escolherá a primeira fonte que suporta.

Tag de áudio

A tag audio é usada para incorporar conteúdo de som em um documento, como música ou outros fluxos de áudio. Essa tag contém uma ou mais source tags com diferentes fontes de áudio. O navegador escolherá a primeira fonte que suporta. Existem três formatos de áudio suportados em HTML: MP3, WAV e OGG.

No YouTube há uma biblioteca de áudio nessa Biblioteca de áudio você encontra produções musicais e efeitos sonoros livres de royalties para usar nos seus vídeos. As músicas e os efeitos sonoros da Biblioteca de áudio não têm restrições de direitos autorais. Esse recurso é exclusivo do YouTube Studio.

Tag Picture

A tag oferece aos desenvolvedores da web mais flexibilidade na especificação de recursos de imagem. O uso mais comum do picture elemento será para direção de arte em designs responsivos. Em vez de ter uma imagem que é dimensionada para cima ou para baixo com base na largura da janela de visualização, várias imagens podem ser projetadas para preencher melhor a janela de visualização do navegador.
